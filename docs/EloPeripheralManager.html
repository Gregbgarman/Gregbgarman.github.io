(static) hasHardwareFeature(feature) → {boolean}

Query if device has hardware feature.
Parameters:
Name Type Description
feature string "pay" - Secure Processor is embedded to support payment service "cell" - LTE/GNSS module is embedded, LTE variant is Europea "cellna" - LTE/GNSS module is embedded, LTE variant is North America "GSen" - Garavity Sensor is supported "LPSen" - Light Sensor and Proxity Sensor is supported "SAR" - SAR Sensor is supported "cuLED" - Customized LED is supported "PwrLED" - Power LED is supported "Batt" - Battery is embedded, chargeing is supported and off-charge mode is supported "eth" - Ethernet is supported "PoE" - Power-over-Ethernet is supported "DuSpk" - Dual Speaker is supported "MoSpk" - Mono Speaker is supported "GPIO" - GPIO port is supported "AdbMux" - ADB multiplexed with other signal, switch is used "eRTC" - external RTC module is used "nqnfc" - NXP NQ310 NFC is supported "vib" - Vibrator is embedded "ZBcr" - Zebra Barcode Scanner Module is supported "exDP" - external DP, comes from DSI2DP Bridge, so that 2-lane-DP + USB-SS can be supported "HDMI" - HDMI port support "DRP" - USB DRP support, USB host & device, PD out & in. "Key3" - support physical key: power, Trigger-left & trigger-right "Key2" - support physical key: power and Home "nCamCSI" - not support MIPI Camera. "amic" - Analog mic-phone is present. "dmic" - Digital mic-phone is present. "SD" - Internal SD card is supported. "CBA" - Customized Boot-Animation is used.

true if hardware feature is supported.

(static) setSLKColor(portsMask, color)

Set the color of Status Light Kit (SLK). If Status Light Kit (SLK) is not enabled, ignore this request directly.
Parameters:
Name Type Description
portsMask int the mask of the port list that you are requesting to update its color, and its new color is provided by param color. mask defined as 1 << 0 is GPIO Port; 1 << 1 is USB Port 1; 1 << 2 is USB Port 2; 1 << 3 is USB Port 3; 1 << 4 is USB Port 4; 1 << 5 is USB Port 5; 1 << 6 is USB Port 6.
color int 1 for Red; 2 for Green.

(static) setSLKPower(portsMask, newStatus)

Request to update power of Status Light Kit (SLK) with ports group If Status Light Kit (SLK) is not enabled, ignore this request directly.
Parameters:
Name Type Description
portsMask int the mask of the port list that you are requesting to update its power, and its new status is provided by param newStatus. mask defined as 1 << 0 is GPIO Port; 1 << 1 is USB Port 1; 1 << 2 is USB Port 2; 1 << 3 is USB Port 3; 1 << 4 is USB Port 4; 1 << 5 is USB Port 5; 1 << 6 is USB Port 6.
newStatus int the power status bits of the port list
