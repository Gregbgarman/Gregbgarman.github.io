Methods

(static) disableZebraBarcode() → {boolean}

Disable Zebra Barcode Scanning. After this API is called (and before {}@link enableZebraBarcode} is called), barcode scanning is disabled using Zebra device.

true iff barcode scanning is disabled.

boolean

boolean
