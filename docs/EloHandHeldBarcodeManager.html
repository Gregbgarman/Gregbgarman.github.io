Class: EloHandHeldBarcodeManager

EloHandHeldBarcodeManager

Class representing Elo HandHeld Barcode Reader Web Service API

Methods

(static) registerHandheldBarcodeListener(func)

Register Callback Function for Elo M50 Hand-Held Barcode Scanner data output.
Parameters:
Name Type Description
func string function name of JavaScript function, that takes an integer and a string for state and data of barcode. State will be != 0 on Barcode read failure. state == 1 means BCR read cancelled; state == 2 means BCR timeout failure.

(static) scanHandheldHandsFree() → {boolean}

Perform Hands-Free (timed) scan for M50 Barcode Scanner. This will emit a laser for the timeout period or until it scans barcode data.
Returns:
true iff hands-free scan successfully started.
Type
boolean

(static) scanHandheldManual() → {boolean}

Perform Manual scan for M50 Barcode Scanner. This will emit a laser for a short period of time or until it scans barcode data.
Returns:
true iff manual scan successfully started.
Type
boolean

(static) setHandheldHandsFreeTimer(millis)

Set M50 Hands-Free scan timeout timer.
Parameters:
Name Type Description
millis int Hand-Free scan timeout timer in Milliseconds

(static) stopHandheldHandsFreeScan() → {boolean}

Stop current Hand Held M50 Hands-Free scan.
Returns:
true iff hand-free scan is stopped.

boolean
